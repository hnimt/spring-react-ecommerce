# Book Ecommerce Application

Ecommerce Website used sale books by entrepreneur

---
**Main Technology**:
1. Server:
  - SpringBoot
  - SpringSecurity
  - JPA, Hibernate
2. Database:
  - MySQL
3. Client:
  - Reactjs
  - Redux, ReduxToolkit
